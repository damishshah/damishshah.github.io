# Damish Shah - Portfolio Website

A portfolio website showcasing my work and experience written in react.

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Running Locally

```bash
npm start
```

The site will be available at `http://localhost:3000`

### Building

```bash
npm run build
```

## Deployment

The site is configured to be served via nginx using Docker Compose. 

1. Build the React application:
```bash
npm run build
```

2. Start the Docker containers:
```bash
docker-compose up -d
```

## Technologies

- React 18
- CSS3
- Docker & Docker Compose
- Nginx
